Useful statistics packages for R for researchers

Useful statistics packages for R for researchers

  • References

Today, I’d like to briefly mention a comprehensive, very practical website that shows how to use R and its extension packages for various statistical procedures as we encounter them in management and sociological research. In a structured way, Mangiafico (2016) presents practical examples, including R code, of every day activities. For example, how to use the likert package of R to work with Likert scale data.

If you have not done so, visit Mangiafico (2016) and get inspired. Thanks!

References

Mangiafico, S. (2016) ‘R handbook’. Available at: http://rcompanion.org/handbook/.
